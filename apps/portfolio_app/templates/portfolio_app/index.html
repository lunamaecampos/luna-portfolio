About Me
I am an artist with a vision to build out projects from the ground up. I am a self-sufficient and resourceful person so I'm not afraid to work on my own. However, I also am open to and value input from my peers and enjoy collaborating ideas with others. My recent endeavors include building out web and iOS applications and stretch to composing/producing musical ideas and planning out small business ventures.

I have taken computer science classes in college and I am a recent graduate of the acclaimed Coding Dojo Bootcamp. I'm currently seeking opportunities to apply my developer skills and build out challenging projects.

L S E MEANSYNTH

*Still in development*

This is a synthesizer web application modeled after a vintage synthesizer from the 80s. The application is built using the MEAN stack and implements the Web Audio API for sound synthesis. Users can sign up and save their favorite patches as JSON data and call back from the database at a later time.

Technologies used: Javascript, MongoDB, ExpressJS, AngularJS, NodeJS, JQuery, Web Audio API, HTML5/CSS3.

X I N X I N

This is a website for the band XINXIN. It's built with RWD to allow it to scale to mobile devices. It features a sign up form that sends out postData to MailChimp service. The application is built using the Django framework. The website admin can sign into a User dashboard and from there can create new tourdates and store them in Django's SQLlite database and delete them too. The tourdates are then displayed on the main band website.

Technologies used: Python, Django, SQLlite, Javascript, JQuery, HTML5/CSS3, RWD.

SYNTH GIRO

This iPhone application is built using the AudioKit framework for sound synthesis. Users can play midi notes with the on-screen keyboard or via midi controller. There is a waveform selection option and modulation of delay or filters are controlled with Apple's CoreMotion Technology which makes for fun expressive play.

Technologies used: Swift, Xcode, AudioKit, CoreMotion, Midi.

B L O S M

*Still in development*

This is a website for a music sharing service aimed towards musicians seeking to collaborate. Users can sign up and easily upload their own music. Media files uploaded are referenced to a separate media folder from the SQLlite database. The media player is HTML/CSS/JS and utilizes the Web Audio API for playback options.
